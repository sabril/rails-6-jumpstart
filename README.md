# Rails 6 Jumpstart
## This repo is to quickly start a Rails 6.1 Project

## The Specs
- Ruby 3.0
- Rails 6.1
- Hotwire
- Devise
- Tailwindcss + TailwindUI
- Alpinejs
- Breadcrumb
- Simple Form

## Install

- Clone the repo
> git clone https://github.com/sabril/rails-6-jumpstart.git your_project_name

> cd your_project_name

> bundle install

> rails db:create

> rails db:migrate

> rails s

- Open https://localhost:3000
